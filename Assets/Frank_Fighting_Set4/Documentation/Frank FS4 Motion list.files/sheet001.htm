Hi I'm Frank,

Thank you for purchasing Fighting Set 4.

This asset was created with my sick body for almost 3 years, and all the motions are hand-made
one by one without reuse. This is why it took so long.

I used a female character, but since there are no completely female motions, it seems that it can be
used for both male and female characters.

I hope this helps game developers, and I hope you make a good game.



Regards,
Frank
